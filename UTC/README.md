# GpsUtcAndLocalTime

GPS，UTC和本地时间的显示器。程序中涉及到朱利安日期的转换，详细介绍见维基百科[Julian day](https://en.wikipedia.org/wiki/Julian_day)。

## 开发环境

* 操作系统：Windows 10 专业版
* IDE：Visual Studio 2015 专业版
